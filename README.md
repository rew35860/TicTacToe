# TicTacToe

Using TKinter, socket, and threading to create the game Tic Tac Toe 

Requirement
--------------------
Python3 

Running the program
--------------------
```
$ Python3 player1.py
```
```
$ Python3 player2.py
```
To get the hostname of your local machine type: 
```
$ hostname
```
For a valid port number, it must be between ``1024 - 65535``
